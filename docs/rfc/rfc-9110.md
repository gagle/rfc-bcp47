# RFC 9110 — HTTP Semantics (Accept-Language)

- **Full title**: HTTP Semantics
- **URL**: https://www.rfc-editor.org/rfc/rfc9110
- **Status**: Internet Standard
- **Relevant sections**: §12.4.2 (Quality Values), §12.5.4 (Accept-Language)

## Scope

RFC 9110 defines HTTP semantics. This library uses only the Accept-Language header parsing rules and quality value syntax for content negotiation.

## Section 12.4.2 — Quality Values

```abnf
weight = OWS ";" OWS "q=" qvalue
qvalue = ("0" [ "." 0*3DIGIT ]) / ("1" [ "." 0*3("0") ])
```

- Decimal number between 0 and 1
- Maximum 3 decimal places
- `1` and `1.000` are equivalent (maximum preference)
- `0` means "not acceptable"
- Default quality is `1.0` when omitted

### Valid Quality Values

| Value | Meaning |
|-------|---------|
| `1` | Maximum preference (same as 1.0) |
| `1.0` | Maximum preference |
| `1.000` | Maximum preference |
| `0.5` | Medium preference |
| `0.001` | Minimal preference |
| `0` | Not acceptable |
| `0.0` | Not acceptable |

### Invalid Quality Values

- `1.1` — exceeds range
- `0.1234` — more than 3 decimal places
- `-0.5` — negative
- `2` — exceeds range

## Section 12.5.4 — Accept-Language

```
Accept-Language = #( language-range [ weight ] )
```

- Comma-separated list of language ranges with optional quality weights
- Language ranges follow RFC 4647 §2.1 (basic language range syntax)
- Used by user agents to indicate natural language preferences
- Enables servers to select appropriate content representation

### Examples

| Header | Parsed result |
|--------|--------------|
| `en` | `[{ tag: 'en', quality: 1.0 }]` |
| `en, fr;q=0.8` | `[{ tag: 'en', quality: 1.0 }, { tag: 'fr', quality: 0.8 }]` |
| `da, en-GB;q=0.8, en;q=0.7` | Sorted by quality descending |
| `*;q=0.5, en;q=1` | `en` first, then `*` |

### Parsing Rules

- Entries are comma-separated
- Whitespace around commas is optional
- Quality parameter follows `;q=`
- Invalid entries should be skipped silently
- Results sorted by quality descending (stable sort preserves insertion order for equal qualities)
